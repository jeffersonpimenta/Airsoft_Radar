# Radar para Airsoft
Script para uso de SDR e RPI para escanear frequências utilizadas pelo rádio Baofeng.

Execute o script e aguarde a voz indicar a frequência detectada, para avançar a frequência pressione uma das teclas.

Utiliza os módulos:
- rtl_power
- rtl_fm
- eSpeak

Utilize pushbuttons nos pinos GND, GPIO26, GPIO19 e GPIO13 para passar as frequências, aumentar e diminuir o threshold para detecção das frequências.
